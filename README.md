# LabPack

Webブラウザで開いて使う、実験や出張で持っていく物をまとめるためのシンプルな HTML ファイルです。

## 使い方

1. `LabPack.html` をブラウザで開きます。
2. 内容を確認・編集して、必要な準備物やチェックリストとして利用します。

## 特徴

- 追加のソフトは不要で、ブラウザだけで利用できます。
- GitHub でバージョン管理ができます。
- 旅行や出張、実験の準備を整理するのに便利です。

## 更新方法

1. ファイルを編集して保存する
2. `git add LabPack.html`
3. `git commit -m "Update LabPack contents"`
4. `git push`

## 補足

必要に応じて、さらに項目やチェックリストを追加して使ってください。
